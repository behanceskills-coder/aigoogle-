GitHub Actions - Daily News Bot Setup Guide
Computer OFF అయినా రోజూ 8AM కి auto run!

STEP 1: GitHub లో New Repository Create చేయండి
1. github.com లో login చేయండి
2. "+" → "New repository" → Name: daily-news-bot → Private → Create

STEP 2: Files Upload చేయండి
- daily_news_bot.py upload చేయండి
- "Create new file" తో .github/workflows/daily_news.yml create చేయండి
  (contents: daily_news.yml file నుండి copy చేయండి)

STEP 3: Gemini API Key (FREE - credit card లేకుండా)
1. https://aistudio.google.com/apikey కి వెళ్ళండి
2. "Create API Key" → copy చేయండి

STEP 4: Google Drive Token (ఒకసారి మాత్రమే - మీ computer లో)
1. https://console.cloud.google.com → New Project "news-bot"
2. APIs → Google Drive API enable చేయండి
3. Credentials → OAuth 2.0 Client ID → Desktop app → credentials.json download
4. మీ computer లో run చేయండి:
   pip install google-auth-oauthlib google-api-python-client
   python get_token.py
5. Browser లో Allow చేయండి → token.json తయారవుతుంది

STEP 5: GitHub Secrets Add చేయండి
Repository → Settings → Secrets → Actions → New secret:
  GEMINI_API_KEY = మీ API key
  GOOGLE_TOKEN   = token.json file contents (అన్నీ copy చేయండి)

STEP 6: Test చేయండి
Actions tab → Daily News Bot → Run workflow → ✅

RESULT:
రోజూ 8AM IST కి Google Drive → "Daily News" folder లో
News_2025-01-15.txt తో 51 AI news summaries!
